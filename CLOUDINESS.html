Cloudiness

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pull data from OPENWEATHERMAP API to assemble a dataset on over 500 cities.

After assembling the dataset, we used MATPLOTLIB to plot various aspects of the weather vs. latitude. Factors we looked at include: Temperature, cloudiness, windspeed, and humidity. This site provides the source data and Visualization created s part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
